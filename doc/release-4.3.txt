SDM4-Bemyndigelseimporter 4.3 Release Notes
=============================

* Fixed NSPSUPPORT-150 ValidTo kolonne må gerne være null + bedre fejlhåndtering når påkrævede felter mangler i xml.

Bemærk denne release indeholder også tidligere database fix hvor BemyndigelsesPid skulle omnavngives til PID - hvis dette allerede er sket udenom applikationen, vil migreringen fejle. Dette kan omgås ved at tilføje følgende til db_migrations tabellen:

insert into db_migrations (version, description, installed_by, installed_on) values ('20121219_1324', 'Bemyndigelse_AlterPIDColumns', 'sdm4', '2013-01-10 11:05:41');

Derved tror migreringsframeworket at filen er kørt på med succes.
